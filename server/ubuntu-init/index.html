Ubuntu 설치 후 패키지 업그레이드, 타임존 설정

ubuntu 22.04 서버 설치 후 기본 설정한 것들 입니다. 패키지를 최신으로 업그레이드 하고 타임존을 Asia/Seoul로 변경합니다.

시스템 업데이트

신규 패키지 설치전에는 먼저 시스템을 최신으로 업데이트하고 시작합니다.

sudo apt update
sudo apt -y upgrade

timezone 설정

date 명령으로 시간을 확인해보면 UTC로 타임존이 설정되어 있습니다. timedatectl 명령으로 Asia/Seoul로 변경합니다.

appaga@appagakr:~$ date
Sat Apr 30 01:04:52 PM UTC 2022
appaga@appagakr:~$ sudo timedatectl set-timezone Asia/Seoul
[sudo] password for appaga:
appaga@appagakr:~$ date
Sat Apr 30 10:06:01 PM KST 2022
