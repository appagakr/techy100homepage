네이버 나눔스퀘어라운드 폰트 적용하기

May 1, 2022

네이버 나눔스퀘어라운드 폰트 적용하기

네이버 hangeul.naver.com에서 제공하는 나눔스퀘어라운드 폰트를 이용해보자. TTF 폰트를 다운받아서 Online Font Convenverter에서 EOT, WOFF, WOFF2 등으로 변환하여 사용하였다. font-nanum.css 파일에 @font-face로 font-family를 정의한다.

+ font-nanum
  font-nanum.css
  + fonts
     NanumSquareRoundB.eot
     NanumSquareRoundB.otf
     NanumSquareRoundB.svg
     NanumSquareRoundB.ttf
     NanumSquareRoundB.woff
     NanumSquareRoundB.woff2
     NanumSquareRoundL.eot
     NanumSquareRoundL.otf
     NanumSquareRoundL.svg
     NanumSquareRoundL.ttf
     NanumSquareRoundL.woff
     NanumSquareRoundL.woff2
     NanumSquareRoundR.eot
     NanumSquareRoundR.otf
     NanumSquareRoundR.svg
     NanumSquareRoundR.ttf
     NanumSquareRoundR.woff
     NanumSquareRoundR.woff2

font-nanum.css 파일 내용 :

@font-face {
  font-family: "NanumSquareRoundR";
  src: url("./fonts/NanumSquareRoundR.eot"); /* IE9 Compat Modes */
  src: url("./fonts/NanumSquareRoundR.eot?#iefix") format("embedded-opentype"), /* IE6-IE8 */
    url("./fonts/NanumSquareRoundR.otf") format("opentype"), /* Open Type Font */
    url("./fonts/NanumSquareRoundR.svg") format("svg"), /* Legacy iOS */
    url("./fonts/NanumSquareRoundR.ttf") format("truetype"), /* Safari, Android, iOS */
    url("./fonts/NanumSquareRoundR.woff") format("woff"), /* Modern Browsers */
    url("./fonts/NanumSquareRoundR.woff2") format("woff2"); /* Modern Browsers */
  font-weight: normal;
  font-style: normal;
}
@font-face {
  font-family: "NanumSquareRoundL";
  src: url("./fonts/NanumSquareRoundL.eot"); /* IE9 Compat Modes */
  src: url("./fonts/NanumSquareRoundL.eot?#iefix") format("embedded-opentype"), /* IE6-IE8 */
    url("./fonts/NanumSquareRoundL.otf") format("opentype"), /* Open Type Font */
    url("./fonts/NanumSquareRoundL.svg") format("svg"), /* Legacy iOS */
    url("./fonts/NanumSquareRoundL.ttf") format("truetype"), /* Safari, Android, iOS */
    url("./fonts/NanumSquareRoundL.woff") format("woff"), /* Modern Browsers */
    url("./fonts/NanumSquareRoundL.woff2") format("woff2"); /* Modern Browsers */

}

    url("./fonts/NanumSquareRoundB.ttf") format("truetype"), /* Safari, Android, iOS */
    url("./fonts/NanumSquareRoundB.woff") format("woff"), /* Modern Browsers */
    url("./fonts/NanumSquareRoundB.woff2") format("woff2"); /* Modern Browsers */
  font-weight: normal;
  font-style: normal;
}

웹페이지 기본 폰트를 NanumSquareRoundR 로 적용한다. 강조가 필요한 요소에는 NanumSquareRoundB를 적용한다.

body {font-family: "NanumSquareRoundR", Georgia, "Times New Roman", serif}

.blog-header-logo {
  font-family: "NanumSquareRoundB", Georgia, "Times New Roman", serif;
  font-size: 2.25rem;
}

h1, h2, h3, h4, h5, h6 {
  font-family: "NanumSquareRoundB", Georgia, "Times New Roman", serif;
}
