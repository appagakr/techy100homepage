alpine 테마로 nuxt 빌드 중 untyped 관련 오류 해결

January 14, 2024

alpine 테마로 nuxt 빌드 중 untyped 관련 오류 해결

alpine 테마를 사용하여 nuxt로 블로그를 만드는 중에 pnpm up -r로 패키지를 업데이트 후 빌드를 실행하였다. 빌드 시에 아래와 같은 오류로 정상적으로 실행되지 않는 상황이 발생하였다.

$ pnpm build
 ERROR  Cannot find module './shared/untyped.a47b2336.cjs'
 Require stack:
- C:\techy100-nuxt-alpine\node_modules\.pnpm\untyped@1.4.1\node_modules\untyped\dist\index.d.cts

untyped: 1.4.1 버전이 문제를 일으키는 것으로 판단되며 버전을 package.json에 1.4.0으로 고정하여 해결할 수 있습니다.

package.json
{
  "name": "techy100-alpine",
  "version": "1.0.0",
  "private": true,
  "scripts": {
    "dev": "nuxi dev",
    "build": "nuxi build",
    "generate": "nuxi generate",
    "preview": "nuxi preview",
    "lint": "eslint ."
  },
  "devDependencies": {
    "@nuxt-themes/alpine": "1.6.5",
    "@nuxt/devtools": "^1.0.8",
    "@nuxt/eslint-config": "^0.2.0",
    "@nuxtjs/plausible": "^0.2.4",
    "@types/node": "^20.11.5",
    "eslint": "^8.56.0",
    "nuxt": "^3.9.3",
    "nuxt-gtag": "^1.1.2"
  },
  "pnpm": {
    "overrides": {
      "untyped": "1.4.0"
    }
  }
}

untyped: 1.4.2 버전에서는 해당 이슈가 해결되었습니다. https://github.com/unjs/untyped/releases/tag/v1.4.2
