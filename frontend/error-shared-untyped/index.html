alpine 테마로 nuxt 빌드 중 untyped 관련 오류 해결

January 14, 2024

alpine 테마로 nuxt 빌드 중 untyped 관련 오류 해결

alpine 테마로 nuxt 빌드 중 untyped, NuxtLink 관련 문제를 수정한 내용입니다.

untyped 관련 Cannot find module 오류 해결

alpine 테마를 사용하여 nuxt로 블로그를 만드는 중에 pnpm up -r로 패키지를 업데이트 후 빌드를 실행하였다. 빌드 시에 아래와 같은 오류로 정상적으로 실행되지 않는 상황이 발생하였다.

$ pnpm build
 ERROR  Cannot find module './shared/untyped.a47b2336.cjs'
 Require stack:
- C:\techy100-nuxt-alpine\node_modules\.pnpm\untyped@1.4.1\node_modules\untyped\dist\index.d.cts

untyped: 1.4.1 버전이 문제를 일으키는 것으로 판단되며 버전을 package.json에 1.4.0으로 고정하여 해결할 수 있습니다.

package.json
{
  "name": "techy100-alpine",
  "version": "1.0.0",
  "private": true,
  "scripts": {
    "dev": "nuxi dev",
    "build": "nuxi build",
    "generate": "nuxi generate",
    "preview": "nuxi preview",
    "lint": "eslint ."
  },
  "devDependencies": {
    "@nuxt-themes/alpine": "1.6.5",
    "@nuxt/devtools": "^1.0.8",
    "@nuxt/eslint-config": "^0.2.0",
    "@nuxtjs/plausible": "^0.2.4",
    "@types/node": "^20.11.5",
    "eslint": "^8.56.0",
    "nuxt": "^3.9.3",
    "nuxt-gtag": "^1.1.2"
  },
  "pnpm": {
    "overrides": {
      "untyped": "1.4.0"
    }
  }
}

untyped: 1.4.2 버전에서는 해당 이슈가 해결되었습니다. https://github.com/unjs/untyped/releases/tag/v1.4.2

NuxtLink의 화면 이동을 처리 오류

alpine 테마를 사용 시에 상단에 네비게이션바가 표시됩니다.

로컬에서는 메뉴 이동이 정상적으로 처리되는데, pnpm generate로 정적 html 파일로 생성하여 서버에 배포하면 링크 url은 브라우저에 변경되어 표시되는데 정작 화면 이동은 되지 않는 문제를 확인했습니다.

\alpine\components\MainNav.vue 파일을 간단히 수정하여 일단 링크가 작동되도록만 조치 했습니다.

MainNav.vue
node_modules\.pnpm\@nuxt-themes+alpine@1.6.5_nuxt@3.9.3_postcss@8.4.33_rollup@3.29.4_vue@3.4.15\node_modules\@nuxt-themes\alpine\components\MainNav.vue

수정한 내용입니다. hanleClick() 함수에 링크를 전달받아서 location.href로 이동하도록 하였습니다. 중요한 것은 끝에 /를 추가해줘야 합니다.

MainNav.vue
function handleClick(link) {
  emits('linkClick')
  console.log('link=', link)
  console.log('location=', location)
  if(link!=='/') {
    location.href = link+'/'
  }
}
</script>

<template>
  <nav>
    <ul>
      <li
        v-for="link of navigation"
        :key="link._path"
      >
        <NuxtLink
          :to="link._path"
          @click="handleClick(link._path)"
        >
          <span class="underline-fx" />
          {{ link.title }}
        </NuxtLink>
      </li>
    </ul>
  </nav>
</template>

이런 문제가 해결된 버전이 빨리 릴리즈 되면 좋겠습니다.
